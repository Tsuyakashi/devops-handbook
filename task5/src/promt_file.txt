Ты — эксперт по анализу логов веб-серверов. Проанализируй предоставленные логи Nginx и выполни следующие задачи:

## Задача 1: Поиск запросов с кодом ответа 400
Найди все запросы в логах, которые имеют HTTP код ответа 400 (Bad Request). Код ответа обычно находится в одном из последних полей строки лога.

## Задача 2: Группировка по IP-адресам
Сгруппируй все найденные запросы с кодом 400 по IP-адресам клиентов. IP-адрес обычно находится в начале строки лога.

## Задача 3: Подсчёт количества запросов
Для каждого уникального IP-адреса подсчитай общее количество запросов с кодом 400. Представь результат в виде таблицы или списка, отсортированного по убыванию количества запросов.

## Задача 4: Анализ проблемных endpoints
Определи наиболее частые endpoints (URI/пути), которые приводят к ошибкам 400. Для каждого endpoint укажи:
- Полный путь/URI
- Количество ошибок 400
- Список IP-адресов, которые генерируют ошибки на этом endpoint
- HTTP метод (GET, POST, PUT, DELETE и т.д.), если доступен

## Задача 5: Выявление аномалий и подозрительных закономерностей

**Формальное определение аномалий:**
Аномалией считается любое отклонение от нормального паттерна поведения, которое может указывать на:
1. **Высокая частота ошибок от одного IP**: IP-адрес генерирует более 10 ошибок 400 за анализируемый период (или более 20% от общего числа ошибок, если общее число меньше 50).
2. **Повторяющиеся паттерны запросов**: Один и тот же endpoint генерирует ошибки 400 от трёх или более разных IP-адресов.
3. **Необычные endpoints**: Запросы к несуществующим или подозрительным путям (например, содержащим попытки SQL-инъекций, path traversal, или известные уязвимости).
4. **Временные кластеры**: Концентрация ошибок 400 в короткий временной интервал (более 5 ошибок за 1 минуту).
5. **Нестандартные методы или заголовки**: Использование необычных HTTP методов или подозрительных User-Agent строк в запросах с ошибками 400.

Опиши найденные аномалии, указав:
- Тип аномалии (из списка выше)
- Конкретные примеры из логов
- Рекомендации по дальнейшему расследованию
